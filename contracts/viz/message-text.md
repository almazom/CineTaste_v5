---

## Обязательные поля

### Корень
- `text` — текст сообщения в Telegram Markdown
- `meta` — метаданные форматирования

### Meta
- `template` — использованный шаблон
- `formatted_at` — время форматирования

---

## Формат сообщения

Сообщение использует Telegram Markdown:
- `*жирный*` → **жирный**
- `_курсив_` → *курсив*
- `[ссылка](url)` → гиперссылка

---

## Пример данных

```json
{
  "text": "🎬 *Фильмы недели* (Москва)\n\n🌟 *Матрица* (1999)\n🎬 Реж: Вачовски\n⭐ Score: 92\n\n[Подробнее](https://kinoteatr.ru/matrix)",
  "meta": {
    "template": "weekly-v1",
    "city_display": "Москва",
    "movie_count": 2,
    "formatted_at": "2024-03-15T12:00:00Z"
  }
}
```

## Аналогия

Это как **открытка**:
- `text` — само содержание открытки
- `meta` — информация о дизайне (шаблон) и параметрах
